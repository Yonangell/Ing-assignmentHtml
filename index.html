¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Se utiliza para estructurar contenido en la web mediante el uso de etiquetas.

Las etiquetas HTML son elementos que definen la estructura y el contenido de una página web. Se componen de una etiqueta de apertura y una etiqueta de cierre, con el contenido en el medio.

Elementos básicos de HTML

  3. Estructura de una página HTML
  • Etiquetas: Son los elementos fundamentales de HTML. Se utilizan para definir la estructura y el contenido de la página.
  • Atributos: Proporcionan información adicional sobre las etiquetas. Se definen dentro de la etiqueta de apertura.
  • Contenido: Es el texto o los elementos que se encuentran dentro de las etiquetas.

Ejemplo de una página HTML

        <!DOCTYPE html>
        <html lang="es">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Ejemplo de página HTML</title>
        </head>
        <body>
            <h1>Bienvenido a mi página HTML</h1>
            <p>Este es un ejemplo simple de una página HTML.</p>
        </body>
        </html>
